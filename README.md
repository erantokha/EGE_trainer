# ЕГЭ-тренажёр: задачи + домашние задания

В этой версии проекта оставлены только:
- тренажёр задач (выбор → тестирование/список)
- создание домашнего задания (для учителя)
- выполнение домашнего задания по ссылке (для ученика)

Удалены (будут переписаны позже): вход с вводом имени, меню, формулы, статистика ученика, админка преподавателя, экспериментальные «фазы».

## Страницы

- / (index.html) — старт: две кнопки
  - «Тренажёр задач» → /tasks/index.html
  - «Создать ДЗ» → /tasks/hw_create.html

- /tasks/index.html — выбор тем/разделов
- /tasks/trainer.html — прохождение теста (открывается из выбора)
- /tasks/list.html — список подобранных задач (открывается из выбора)
- /tasks/hw_create.html — создание ДЗ (требует вход учителя через Google/Supabase)
- /tasks/hw.html?token=... — выполнение ДЗ учеником

## Контент

Все задания лежат в папке content/.
Главный каталог: content/tasks/index.json.

## Развёртывание

Это статический фронтенд без сборки: достаточно GitHub Pages (или любого статического хостинга).
Важно: файл .nojekyll должен присутствовать (он уже в корне).
