
# Навигация по ключевым файлам (L2)

Эта секция отвечает на вопрос “в каком файле менять X”.

Оглавление
- app/ (конфиг, build, ядро)
- app/providers (Supabase, запись попыток, ДЗ)
- app/ui (шапка)
- tasks/auth (вход, callback, reset, профиль)
- tasks/trainer (выбор тем, тренажёр, список, уникальные)
- tasks/homework (создание и выполнение ДЗ)
- tasks/stats (статистика и умный режим)
- tasks/teacher (ученики и карточка ученика)
