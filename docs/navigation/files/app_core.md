
# app/core (L2)

## ../../../app/core/pick.js

Путь: app/core/pick.js

Назначение: алгоритмы выбора набора задач из нескольких тем/подтем.

Ответственность:
- делает: выбор k задач, перемешивание, равномерное распределение по базам/подтемам
- не делает: не грузит json, не рендерит DOM, не пишет в Supabase

Как подключается:
- импортируется модулями “умного режима” и сборки “умного ДЗ” в tasks/

Публичный API (типично):
- sampleKByBase(items, k, baseKey)
- interleaveBatches(batches)
- shuffleInPlace(arr)

Зависимости:
- внеш.: Math.random (важно для воспроизводимости, если нужен seed)

Побочные эффекты:
- изменяет массивы на месте (shuffleInPlace)

Тонкости/риски:
- если “уникальность” задач важна, нужно следить за устранением дублей до/после выборки
