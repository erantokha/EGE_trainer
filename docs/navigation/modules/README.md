# Карта модулей и папок (L1)

Цель этой секции: быстро понять, где что лежит по папкам, и куда идти за нужной фичей.

Топ‑уровень

- [root](./root.md) — точка входа, ключевые файлы в корне, общие конвенции
- [.github](./github.md) — CI, проверки и workflow
- [app](./app.md) — общие модули (конфиг, Supabase, утилиты, общий UI)
- [tasks](./tasks.md) — страницы приложения (тренажёр, ДЗ, статистика, авторизация)
- [content](./content.md) — банк задач (JSON) и изображения

Крупные подпапки

- [app/core](./app_core.md) — алгоритмы выбора задач и вспомогательные утилиты “ядра”
- [app/providers](./app_providers.md) — клиент Supabase и провайдеры данных (attempts, homework)
- [app/ui](./app_ui.md) — общий UI (шапка и навигация)

Фичи внутри tasks

- [tasks/auth](./tasks_auth.md) — вход/регистрация/колбэки/восстановление пароля
- [tasks/trainer](./tasks_trainer.md) — выбор тем, тренажёр, список задач
- [tasks/homework](./tasks_homework.md) — создание ДЗ, выполнение ДЗ, экраны результата
- [tasks/stats](./tasks_stats.md) — статистика ученика, рекомендации, умный режим
- [tasks/teacher](./tasks_teacher.md) — кабинет учителя и страницы ученика

Контент

- [content/tasks](./content_tasks.md) — структура JSON‑манифестов тем и правил именования
- [content/topics](./content_topics.md) — альтернативные наборы вопросов (наследие/эксперименты)
- [content/img](./content_img.md) — изображения, правила размещения и ссылки из задач
