# content/img: изображения (L1)

Назначение

- изображения/схемы, используемые задачами
- пути на изображения обычно прописаны в `figure.img` внутри манифестов тем

Где используются

- тренажёр:
  - [tasks/trainer.js](../../../tasks/trainer.js) рендерит figure.img
- домашние задания:
  - [tasks/hw.js](../../../tasks/hw.js) рендерит figure.img
- просмотр уникальных:
  - [tasks/unique.js](../../../tasks/unique.js)

Правила размещения

- складывать изображения в [content/img/](../../../content/img/)
- в JSON указывать путь вида `content/img/<name>.<ext>`
- не переименовывать уже используемые файлы без поиска по репозиторию

Точки расширения

- можно вводить подкаталоги внутри content/img, если:
  - они не ломают существующие ссылки
  - пути в манифестах обновлены согласованно
