# content/topics: альтернативные наборы вопросов (L1)

Назначение

- дополнительные/экспериментальные форматы хранения вопросов
- на текущий момент основной банк задач живёт в [content/tasks/](../../../content/tasks/)

Состав

- [content/topics/](../../../content/topics/)
  - внутри встречаются `questions.json` и связанные ассеты

Как относиться

- если вы добавляете новые задачи для тренажёра/ДЗ:
  - используйте формат [content/tasks/](./content_tasks.md)
- если нужно мигрировать questions.json в основной банк:
  - сделать конвертер в отдельном скрипте (вне фронта)
  - после миграции добавить тему в [content/tasks/index.json](../../../content/tasks/index.json)

Риски

- смешивание двух форматов повышает вероятность рассинхронизации статистики (question_id)
- если какая‑то страница внезапно зависит от content/topics, это нужно явно зафиксировать в документации
