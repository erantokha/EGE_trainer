# Папка app/core (L1)

Назначение

- алгоритмы выбора задач из банка по темам/подтемам
- утилиты для перемешивания и “справедливого” распределения

Состав

- [app/core/pick.js](../../../app/core/pick.js)
  - выбор k задач по “базам” (подтемам) и перемешивание
  - используется в “умных” режимах сборки списка задач/ДЗ

Кто использует

- страницы в [tasks/](../../../tasks/) через импорты из “умного режима”:
  - [tasks/smart_select.js](../../../tasks/smart_select.js)
  - [tasks/smart_hw_builder.js](../../../tasks/smart_hw_builder.js)
  - [tasks/smart_hw.js](../../../tasks/smart_hw.js)

Точки расширения

- новый алгоритм подбора:
  - добавить функцию в pick.js или новый модуль рядом
  - обновить вызовы в smart_* файлах

Тонкости/риски

- стабильность question_id: алгоритмы выбирают существующие id из JSON‑манифестов, поэтому любые переименования/удаления требуют аккуратности
- “случайность” должна быть воспроизводимой для ДЗ, если используется seed (см. [tasks/hw_create.js](../../../tasks/hw_create.js))
