
# Глоссарий

- прототип: шаблон задачи в JSON-манифесте темы
- аналог: вариант прототипа (если используется генерация/вариативность)
- unic: флаг “уникальный прототип”, используется для выборок/проверок
- attempt: попытка решения задачи вне ДЗ (таблица attempts)
- homework: домашнее задание (таблица homeworks)
- homework_link token: токен ссылки на ДЗ (таблица homework_links)
- homework_attempt: попытка выполнения ДЗ (таблица homework_attempts)
- answer_event: унифицированное событие ответа для статистики (таблица answer_events)
- RLS: Row Level Security, политики доступа на таблицах в Postgres
- RPC: вызов функции Postgres через PostgREST (/rest/v1/rpc/...)
- PKCE: механизм OAuth, используемый supabase-js для безопасного обмена code → session
