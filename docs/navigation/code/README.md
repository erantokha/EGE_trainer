# Снимки кода (для чтения документации отдельно от репозитория)

Эта папка содержит копии ключевых файлов, чтобы документация оставалась читаемой в формате docs.zip.
В репозитории источником истины остаётся код в корневых папках `app/` и `tasks/`.

Содержимое
- [app/providers/supabase.js](./app/providers/supabase.js)
- [app/providers/homework.js](./app/providers/homework.js)
- [app/providers/supabase-write.js](./app/providers/supabase-write.js)
- [app/ui/header.js](./app/ui/header.js)
- [tasks/auth.js](./tasks/auth.js)
- [tasks/auth_callback.js](./tasks/auth_callback.js)
- [tasks/hw.js](./tasks/hw.js)
- [tasks/hw_create.js](./tasks/hw_create.js)
- [tasks/stats.js](./tasks/stats.js)
- [tasks/my_students.js](./tasks/my_students.js)

Как обновлять
- при изменениях в коде пересохранить соответствующие файлы в этой папке
- обновить дату в docs/navigation/README.md и запись в docs/navigation/changelog.md
