## 1. Что хотим получить в итоге (целевая логика — для контекста разведки)

- **Прототип:** точность = верные из **последних 3 попыток** (3 самых свежих ответа по времени, по всем вопросам прототипа). <3 попыток — по факту; 0 — нет данных.
- **Подтема %** = **среднее процентов** её прототипов (с данными).
- **Тема %** = **среднее процентов** её подтем.
- **Бейджи** (red<50/yellow<70/lime<90/green≥90) — по этим last-3-процентам.
- **Итоговые баллы** (уже реализованы: номер/тема = 1 балл × доля → сумма первичных → перевод во вторичные) — должны опираться на **те же last-3 проценты**.
- **Не сломать** работающий скоринг; ввести «последние 3» как сущность (возможно, она частично уже есть в БД).
